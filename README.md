# Plano B Analytics 📊

Projeto de **Business Intelligence aplicado a um cenário real de negócio**, desenvolvido para análise de dados da **Plano B Tabacaria**.

O objetivo do projeto é construir uma estrutura completa de dados utilizando **MySQL, SQL e Power BI**, permitindo o acompanhamento de vendas, estoque, movimentações e indicadores estratégicos para tomada de decisão.

---

## 🎯 Objetivo do Projeto

Criar uma solução de análise de dados capaz de transformar informações operacionais em insights de negócio, simulando um ambiente real utilizado por empresas para controle e crescimento.

O projeto contempla:

* Modelagem e estruturação de banco de dados;
* Organização de dados de produtos, vendas e estoque;
* Criação de consultas analíticas em SQL;
* Desenvolvimento de indicadores de desempenho;
* Preparação dos dados para visualização no Power BI.

---

## 🛠️ Tecnologias Utilizadas

* **MySQL** — Banco de dados relacional;
* **SQL** — Criação de tabelas, consultas e análises;
* **Power BI** — Visualização e criação de dashboards;
* **Git/GitHub** — Versionamento e documentação do projeto.

---

## 🗄️ Estrutura do Banco de Dados

O banco de dados foi desenvolvido seguindo boas práticas de organização e relacionamento entre entidades.

Principais tabelas:

* `produtos` — Cadastro dos produtos comercializados;
* `vendas` — Registro das operações de venda;
* `itens_venda` — Detalhamento dos produtos vendidos;
* `compras` — Controle de entradas de produtos;
* `movimentacoes_estoque` — Histórico de movimentação do estoque.

---

## 📈 Análises Desenvolvidas

O projeto possui consultas analíticas para:

* Ranking de produtos vendidos;
* Controle de estoque atual;
* Análise de movimentações;
* Acompanhamento de vendas;
* Geração de indicadores para tomada de decisão.
